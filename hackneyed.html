Hackneyed kernel for Asus Zenfone Max Pro M1 (X00TD/X00T)

Hackneyed is a custom HMP kernel which mainly concentrates on balanced profile. It is a pure caf kernel with some tweaks and modifications. The kernel is not overclocked or underclocked. It maintains the stock frequencies. If u guys need maximum performance on a battery balanced kernel, give it a try. Have a nice day.
