LineageOS for Realme 9 Pro 5G/Realme Q5 (RMX3471/RMX3472/RMX3478)

LineageOS is an operating system for smartphones, tablet computers, and set-top boxes, based on Android with mostly free and open-source software. It is the successor to the custom ROM CyanogenMod, from which it was forked in December 2016 when Cyanogen Inc. announced it was discontinuing development and shut down the infrastructure behind the project. Since Cyanogen Inc. retained the rights to the Cyanogen name, the project rebranded its fork as LineageOS.

Changelogs:

* Updated blobs from RMX3471_11.C.21
* Kernel upsteamed to 5.4.259
* Merged December security patch
* Read notes before flashing.

Notes:

* Temporarily dropped support for CN variant
* Need to perform clean flash
* Use the mentioned recovery to install
* Take a backup of your data as this process will erase /data partition
* If you face any issues related to sensors after flashing this build, u need to reflash stock and come back to this build (happens bcz of kernel and fw mismatch) or u can try flashing c10 modem (Most of the time, that fixes too)
* Follow instructions carefully for flashing
* MindTheGapps preferred for gapps (can try others too, i haven't tested those)
* In case if you find any bugs, report with proper logs
* Charging control hal might send a notification which cant be cleared by swiping. If happens so, just long press on the notification and turn off notifications for charging hal.

How to Flash:

* Download all 4 files listed above (Rom, boot, dtbo, vendor_boot)
* Reboot to bootloader
* Type the following three commands carefully from the directory where the files are downloaded
* fastboot flash boot boot.img
* fastboot flash dtbo dtbo.img
* fastboot flash vendor_boot vendor_boot.img
* After flashing all three files, reboot to recovery
* Format data (ignore if recovery complains about metadata type)
* Select apply update > install using adb
* Connect device to computer and type the following command
* adb sideload lineage***.zip (Replace with the filename of the zip)
* After flash completed, recovery will ask to reboot to recovery if in case u need to flash any other zips like magisk, gapps etc. Proceed accordingly.
* Format data once again after everything completed
* Now you successfully flashed the rom. Boot into the rom and enjoy.
